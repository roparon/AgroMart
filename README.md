# 🌾 AgroMart

**AgroMart** is a modern, full-featured e-commerce platform for agricultural machinery, industrial equipment, refrigeration products, and genuine spare parts. Built with **Python (Flask)**, **HTML5**, **CSS3**, **JavaScript**, and **SQL**, the platform is designed to provide farmers, businesses, and individuals with a convenient way to browse, compare, and purchase high-quality equipment online.

---

## 📖 About the Project

AgroMart aims to simplify the purchase of agricultural and industrial equipment by providing a secure, responsive, and easy-to-use online shopping experience.

Customers can browse products by category, search for equipment, add items to a shopping cart, place orders, and manage their accounts. Administrators can manage inventory, customers, orders, and product categories through a dedicated dashboard.

## 🚀 Installation

### Clone the repository

```bash
git clone <repository-url>
cd AgroMart
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the virtual environment

Linux/macOS

```bash
source venv/bin/activate
```

Windows

```bash
venv\Scripts\activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python run.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
